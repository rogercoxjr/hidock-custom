# Contract Coverage Matrix

Coverage ledger for the renderer REST SDK groups (`src/lib/electron-api/groups/*.ts`) against
the Layer-2 SDK↔route contract tests (`src/lib/electron-api/__tests__/*.contract.test.ts`).

For every group method this records: the **HTTP verb + path** the SDK call actually issues, the
**Fastify route file** that backs that path (`electron/server/routes/*.ts`), and whether a
**contract test** currently exercises it.

> This is a documentation artifact only — there is no test to run. It is a point-in-time snapshot
> generated by reading the SDK group sources, the route registrations, and the contract-test
> files. Re-derive it if any of those change.

## Legend

| Mark | Meaning |
| --- | --- |
| ✅ | Covered — a `*.contract.test.ts` exercises this method end-to-end (SDK → real Fastify route). |
| ❌ | **NOT COVERED** — no contract test hits this method (a real gap). |
| ⏭️ | Intentionally skipped with a documented reason in the group's contract test (live network / LLM / multipart). Still uncovered, but by design. |
| N/A | Not a REST↔route contract — no Fastify route backs it (WebUSB device call, WebSocket subscription, or a client-side `DROPPED` no-op). Out of scope for contract tests by construction. |

## Route-file cross-references (non-obvious mappings)

Routes have **no shared prefix** (`app.register` per file, full paths inline), so several SDK
methods hit a route file whose name does not match the SDK group:

- `recordings.*` transcription/queue methods (`transcribe`, `resummarize`, `cancelTranscription`,
  `getTranscriptionStatus`, `retryAllFailed`, `processQueue`, `updateQueueItem`, `isSummaryStale`,
  `getTranscriptionQueue`, `validateTranscriptionConfig`) → **transcripts.ts** (queue + transcription
  routes live there, not in `recordings.ts`).
- `queue.getItems` (`GET /api/queue`) → **transcripts.ts**.
- `quality.getByQuality` (`GET /api/recordings?quality=`) → **recordings.ts**.
- `recordings.getForMeeting` (`GET /api/meetings/:id/recordings`) → **meetings.ts**.
- `contacts.getForMeeting` (`GET /api/meetings/:id/contacts`) → **contacts.ts**.
- `projects.tagMeeting/untagMeeting/getForMeeting` (`/api/meetings/:meetingId/projects...`) → **projects.ts**.
- `actionables.getByMeeting` (`GET /api/meetings/:id/actionables`) → **actionables.ts**.
- `outputs.getByActionableId` (`GET /api/actionables/:id/output`) → **outputs.ts**.
- `voiceprints.listForContact` (`GET /api/contacts/:contactId/voiceprints`) → **voiceprints.ts**.
- `summarizationTemplates.resummarizeWithTemplate` (`POST /api/recordings/:id/resummarize`) → **transcripts.ts**.
- `summarizationTemplates.latestRun/previewSelection/acceptSuggestedTemplate` (`/api/recordings/:id/template-*`) → **summarization-templates.ts**.
